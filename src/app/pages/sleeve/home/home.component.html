Nuestra Tecnología

La plataforma tecnológica utilizada para los productos de Soluciones Integrales WT es de punta, esta basada en productos de desarrollo lideres en el mercado para moviles y web.

La plataforma de desarrollo es Angular, Java, Nodejs, PWA, entre otros productos destacados para la produccion de software.

Contamos con sistemas para ser utilizados en dispositivos moviles basados en Android, IPhone, o Windows Phone de igual manera para computadores de escritorio o portatiles, asi como nuestro portal web para ser utilizado desde cualquier navegador de internet.
